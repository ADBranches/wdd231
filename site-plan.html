MovieCatalog Site Plan

WDD 231 Individual Project

1. Site Name

MovieCatalog

Description: A modern, intuitive platform for discovering and organizing movies. The name clearly indicates the site's purpose as a comprehensive movie database and organizational tool.

Domain Availability: moviecatalog.app (available) or movie-catalog.org (available)

2. Site Purpose

MovieCatalog provides users with comprehensive movie discovery and organization capabilities:

  • Comprehensive Movie Discovery: Browse thousands of films with advanced filtering capabilities
  • Personal Collections: Create and manage custom movie collections and favorites
  • Detailed Information: Access movie details, ratings, cast information, and metadata
  • Community Features: User-driven movie suggestions and recommendations
  • Advanced Search: Filter by genre, year, rating, popularity, and more

3. Scenarios

These scenarios represent questions from our target audience that drive the website content and functionality:

Scenario 1: Academic Research

User: "I'm a film student looking for highly-rated drama films from the 1990s for my research project. How can I quickly find and organize these movies?"

Solution: Advanced filtering by genre, decade, and rating with collection creation features.

Scenario 2: Social Planning

User: "My friends are coming over for movie night and we can't decide what to watch. Is there a way to filter movies by genre, rating, and release year to find something we'll all enjoy?"

Solution: Multi-criteria filtering and discovery features for group viewing decisions.

Scenario 3: Community Contribution

User: "I noticed a great indie film missing from your catalog. How can I suggest it for addition to help other users discover it?"

Solution: Integrated movie suggestion form with user submission tracking.

Ratings, Highlights, Important Indicators

Color Applications:

  • Primary Blue: Main navigation, primary call-to-action buttons, active links
  • Dark Blue: Button hover states, secondary interactive elements
  • Dark Gray: Header and footer backgrounds for contrast
  • Light Gray: Main content background, card backgrounds
  • White: Card content, text on dark backgrounds
  • Accent Yellow: Star ratings, highlights, warning indicators

5. Typography

Primary Font Family: Segoe UI, Tahoma, Geneva, Verdana, sans-serif

Heading 1 - Main Page Titles (Segoe UI, Semi-bold)

Heading 2 - Section Headers (Segoe UI, Semi-bold)

Heading 3 - Subsection Headers (Segoe UI, Semi-bold)

Body Text - Primary content and paragraphs (Segoe UI, Regular). This font provides excellent readability across all devices and maintains clarity at various sizes.

Navigation & Buttons - Interactive elements (Segoe UI, Medium)

Font Applications:

  • Headings (h1-h3): Segoe UI, semi-bold weight for clear visual hierarchy
  • Body Text: Segoe UI, regular weight for optimal reading experience
  • Navigation & Buttons: Segoe UI, medium weight for better visibility and interaction
  • Fallback Stack: Tahoma, Geneva, Verdana, sans-serif for cross-platform compatibility

7. Technical Specifications

HTML & CSS Validation

  • HTML5: Semantic structure with proper header, main, section, footer elements
  • CSS3: Modern features including custom properties, flexbox, and CSS grid
  • Accessibility: ARIA labels, proper heading hierarchy, keyboard navigation
  • Responsive Design: Mobile-first approach with breakpoints at 768px and 1024px

Project Structure

  • Homepage (index.html): Featured movies, popular collections, search
  • Movie Catalog (catalog.html): Advanced filtering, grid/list views
  • Collections (collections.html): User-created movie collections
  • Attributions (attributions.html): Resource credits and technologies
